About the Styleguide

These are some very basic style and branding guidelines to help give OpenTechSchool a strong and consistent visual appearance. We are aiming to extend and improve the styleguide over time and make it a useful tool and resource center for all OTS members working on our OTS websites, print product, presentations etc.

Name and Logo

OpenTechSchool is spelled as one word, in CamelCase. If used in connection with workshop titles, events etc, just add OpenTechSchool, without anything like proudly presents or presented by.

The logo comes in two versions, a one-line and a two-line version. Ideally the logo is used in OTS blue, or in black for b/w printing. Another option are inverted colors. Please always allow some whitespace around the logo to make it stand out from other elements.

Colors

  • #085987 #085987 OTS blue: Use for logo
  • #1F8AB6 #1F8AB6 OTS blue link: Use for links
  • #2FB0D5 #2FB0D5 OTS blue hover: Use for blue hover states
  • #222222 #222222 OTS text: Use for default text color
  • #E04C3E #E04C3E OTS red: Use for highlighted links, buttons, other highlights
  • #FD4B22 #FD4B22 OTS red hover: Use for red hover states

Page Layout

All websites under the OTS umbrella should use the same basic layout, that consists of: A blue header and footer, and a white main content area.

The up-to-date templates can be found here: OTS on Github

More templates for workshop pages etc. will be provided soon. Please let us know what you need!
